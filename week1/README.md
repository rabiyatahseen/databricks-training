# Week 1 - Databricks SQL Training

## 📌 Overview

This repository contains my Week 1 work for the Databricks and SQL training program. During this week, I practiced SQL fundamentals, joins, analytical queries, window functions, and Common Table Expressions (CTEs) using DB Fiddle and GitHub.

The main goal of this training is to build strong SQL problem-solving skills and become comfortable working with relational databases and analytical queries.

# 🛠️ Tools and Platforms Used

* DB Fiddle for SQL query execution and practice
* GitHub for repository management and version control
* PostgreSQL and MySQL for database practice
