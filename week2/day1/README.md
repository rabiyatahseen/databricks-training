# Week 2 - Day 1: College Management System SQL Practice

## 📌 Overview

On Day 1 of Week 2, I started practicing SQL using a College Management System (CMS) database. The focus of today's work was understanding relational database structure and solving queries using joins, aggregate functions, filtering, and grouping operations.

## 🛠️ Tools Used

* DB Fiddle (PostgreSQL) for SQL execution and testing
* GitHub for maintaining and organizing work

## 📚 Topics Covered

* SQL Query Writing using PostgreSQL and DB Fiddle
* INNER JOIN and LEFT JOIN
* Aggregate Functions and Analytical Queries
* GROUP BY and HAVING Clauses
* DISTINCT and Conditional Filtering
* NULL Handling using `COALESCE()`
* String Functions

  * `UPPER()`, `LOWER()`, `INITCAP()`
  * `LENGTH()`, `LEFT()`, `RIGHT()`
  * `SUBSTRING()`, `REPLACE()`
* Date and Time Functions

  * `CURRENT_DATE`
  * `EXTRACT()`
  * `AGE()`
  * `TO_CHAR()`
  * `DATE_TRUNC()`
* Numeric Functions

  * `ROUND()`, `TRUNC()`
  * `FLOOR()`, `CEIL()`
  * `ABS()`, `MOD()`, `SIGN()`
  * `POWER()`
* CASE Statements and Conditional Logic
* Timestamp and Working Hour Calculations
* Weekday / Weekend Detection Logic
* PostgreSQL Type Casting and Error Handling
* Analytical and Reporting Queries
* Real-world Business SQL Problem Solving
