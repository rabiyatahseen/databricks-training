## 🗂️ Repository Structure

The repository is organized in a **week-wise and day-wise format** to maintain clarity and track progress effectively.

```
databricks-training/
 └── week1/
      ├── README.md
      ├── day1/
      │    ├── README.md
      │    ├── queries.sql
      │    └── output.txt
      └── day2/
```

Each **day folder** contains:

* SQL query files
* Output/results of executed queries
* A README explaining daily learning
